# قوالب التقارير PDF

هذا المجلد يحتوي على قوالب PDF للتقارير الطلابية.

## التعليمات:

1. ضع ملف PDF القالب هنا
2. اسم الملف يجب أن يكون: `student_report_template.pdf`
3. بعد وضع الملف، سيتم استخدامه كقالب لإضافة البيانات عليه

## ملاحظات:

- يجب أن يكون القالب بصيغة PDF
- يمكن استخدام مكتبة مثل `pdf-lib` أو `jspdf` لإضافة البيانات على القالب
- البيانات التي سيتم إضافتها:
  - معلومات الطالب (الاسم، الفصل)
  - معلومات المدرسة (الاسم، الشعار، العنوان)
  - تاريخ التقرير
  - الحضور (حاضر/غائب/مستأذن)
  - المشاركة (متميز/جيد/متوسط/ضعيف)
  - الواجبات (متميز/جيد/متوسط/ضعيف)
  - السلوك (متميز/جيد/متوسط/ضعيف)
  - الملاحظات
  - الجدول اليومي
