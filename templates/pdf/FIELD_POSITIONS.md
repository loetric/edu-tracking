# إعداد مواضع الحقول في قالب PDF

## التعليمات:

بعد فتح ملف PDF القالب، يجب تحديد إحداثيات (x, y) لكل حقل في القالب.

### كيفية تحديد الإحداثيات:

1. افتح ملف PDF القالب في برنامج مثل Adobe Acrobat أو PDF Editor
2. استخدم أداة القياس أو أدوات التعليقات لتحديد موضع كل حقل
3. في PDF، الإحداثيات تبدأ من الزاوية السفلية اليسرى (0, 0)
4. الوحدة المستخدمة هي النقاط (Points) حيث 1 بوصة = 72 نقطة

### الحقول المطلوبة:

#### معلومات الطالب:
- `studentName`: اسم الطالب
- `studentClass`: الفصل
- `studentId`: رقم الملف/الطالب
- `parentPhone`: جوال ولي الأمر

#### التاريخ:
- `reportDate`: تاريخ التقرير
- `dayName`: اسم اليوم

#### معلومات المدرسة:
- `schoolName`: اسم المدرسة
- `ministry`: الوزارة
- `region`: الإدارة التعليمية
- `schoolPhone`: رقم واتساب المدرسة

#### الحضور والأداء:
- `attendance`: الحضور (حاضر/غائب/مستأذن)
- `participation`: المشاركة
- `homework`: الواجبات
- `behavior`: السلوك

#### ملاحظات:
- `notes`: الملاحظات

#### الجدول:
- `schedule`: مصفوفة من المواضع للجدول اليومي (إذا كان هناك عدة صفوف)

### مثال على التكوين:

```typescript
const customFields: Partial<TemplateFields> = {
  studentName: { x: 100, y: 700, fontSize: 14, font: 'bold' },
  studentClass: { x: 100, y: 680, fontSize: 12, font: 'regular' },
  reportDate: { x: 400, y: 700, fontSize: 12, font: 'regular' },
  attendance: { x: 100, y: 600, fontSize: 12, font: 'regular' },
  // ... إلخ
};
```

### ملاحظات:

- `x`: المسافة من اليسار (بالنقاط)
- `y`: المسافة من الأعلى (بالنقاط) - سيتم تحويلها تلقائياً
- `fontSize`: حجم الخط (افتراضي: 12)
- `font`: نوع الخط ('regular' أو 'bold')
- `color`: لون النص (افتراضي: أسود)
- `width`: العرض الأقصى للنص (للتفاف النص)

### بعد تحديد المواضع:

1. افتح ملف `services/pdfGenerator.ts`
2. ابحث عن `DEFAULT_FIELDS`
3. قم بتحديث الإحداثيات بناءً على قالبك
4. أو استخدم `customFields` عند استدعاء الدالة
